# YuSeongChoi.github.io

Root GitHub Pages site used for files that must be served from `https://yuseongchoi.github.io/`, including AdMob `app-ads.txt`.

AdMob checks the hostname from the App Store marketing URL, so Harmony of Stars uses:

```text
https://yuseongchoi.github.io/app-ads.txt
```

Replace the placeholder in `app-ads.txt` with the personalized snippet from AdMob.
